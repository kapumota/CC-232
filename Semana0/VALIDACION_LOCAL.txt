### Validacion local de Semana0

#### Objetivo

Este archivo resume los comandos minimos para verificar Semana0 desde la raiz del
repositorio.

#### Compilar Proyecto0

```bash
rm -rf build_semana0
cmake -S Semana0/Proyecto0 -B build_semana0
cmake --build build_semana0
```

#### Ejecutar pruebas

```bash
ctest --test-dir build_semana0 --output-on-failure
```

#### Ejecutar demos principales

```bash
./build_semana0/demo_line_stats
./build_semana0/demo_const_refs
./build_semana0/demo_copy_semantics
./build_semana0/demo_raii
```

En Windows con generadores multi configuracion, los ejecutables pueden quedar en
una carpeta como `build_semana0/Debug`.

#### Limpiar artefactos

```bash
rm -rf build_semana0
rm -rf Semana0/Proyecto0/Testing
rm -rf Semana0/ejercicios0_out
```

#### Validacion de documentacion

```bash
grep -RIn "====\|—\|→" Semana0/*.md Semana0/lecturas/*.md Semana0/avanzado/*.md || true
```

El comando anterior ayuda a encontrar separadores o simbolos que no se quieren en
la documentacion del curso.
