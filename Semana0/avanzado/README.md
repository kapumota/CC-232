### Material avanzado de Semana0

#### Proposito

Esta carpeta contiene experimentos que apoyan el curso, pero que no deben mezclarse con la primera validacion tecnica del estudiante.

#### Contenido

| Archivo | Proposito |
|---|---|
| `Ejercicios0_experimentos_gpp.md` | Experimentos con optimizacion, sanitizers, cobertura, profiling y STL. |
| `INSTRUCCIONES_Ejercicios0_v4_2.md` | Guia de ejecucion del script de experimentos. |

#### Criterio de uso

El material avanzado se usa cuando el estudiante ya puede:

1. Compilar `Proyecto0` con CMake.
2. Ejecutar pruebas con CTest.
3. Limpiar artefactos generados.
4. Distinguir entre correctitud, medicion y optimizacion.

#### Temas que se mantienen como avanzados

1. LTO.
2. PGO.
3. Profiling con `gprof`.
4. Cobertura con herramientas externas.
5. Warnings dependientes de optimizacion.
6. Experimentos comparativos con STL.

| `Optimizacion-LTO-PGO.md` | Optimización LTO y PGO. |
| `Profiling-gprof.md` | Profiling básico con gprof. |
| `Cobertura-Avanzada.md` | Cobertura de líneas, ramas y funciones. |
