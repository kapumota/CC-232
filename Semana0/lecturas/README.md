### Lecturas de la Semana0

#### Propósito

Esta carpeta contiene las lecturas obligatorias de preparación técnica para CC232.

Las lecturas están pensadas para que el estudiante pueda compilar, probar, depurar y razonar sobre código C++17 antes de implementar estructuras de datos propias.

#### Lecturas actuales

| Archivo                            | Fase     | Propósito                                                                 |
| ---------------------------------- | -------- | ------------------------------------------------------------------------- |
| `Lectura00-Cpp-Minimo-CC232.md`    | Fase 0.2 | Introduce la base de C++17 necesaria antes de estructuras de datos.       |
| `Lectura01-Templates-Genericos.md` | Fase 0.3 | Introduce funciones y clases genéricas para estructuras de datos.         |
| `Lectura02-CMake-CTest-CC232.md`   | Fase 0.4 | Explica CMake moderno, targets, opciones de compilación y CTest.          |
| `Lectura03-Pruebas-Unitarias.md`   | Fase 0.1 | Introduce pruebas unitarias, `assert`, CTest y validación de contratos.   |
| `Lectura04-Sanitizers.md`          | Fase 0.5 | Introduce ASan, UBSan y el flujo de depuración con sanitizers.            |
| `Lectura06-STL-Complejidad.md`     | Fase 0.6 | Presenta STL, iteradores, algoritmos estándar, adaptadores y complejidad. |

#### Regla de uso

Las lecturas de esta carpeta forman parte del flujo obligatorio del curso.

El estudiante debe revisarlas antes de avanzar a las estructuras de datos principales, porque establecen el entorno técnico, las reglas de validación, el uso básico de C++17, las pruebas, los sanitizers y el uso razonado de STL.

#### Material avanzado

Las lecturas de `Semana0/avanzado` se usan después de que el estudiante ya puede compilar, probar, depurar y limpiar el proyecto base.

Ese material no debe mezclarse con la primera validación técnica de Semana0.
