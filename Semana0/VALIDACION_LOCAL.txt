### Validación local de Semana0

#### Objetivo

Este archivo resume los comandos mínimos para verificar Semana0 desde la raíz del repositorio.

#### Compilar Proyecto0

```bash
rm -rf build_semana0
cmake -S Semana0/Proyecto0 -B build_semana0
cmake --build build_semana0
```

#### Ejecutar pruebas

```bash
ctest --test-dir build_semana0 --output-on-failure
```

#### Ejecutar demos principales

```bash
./build_semana0/demo_line_stats
./build_semana0/demo_const_refs
./build_semana0/demo_copy_semantics
./build_semana0/demo_raii
```

En Windows con generadores multiconfiguración, los ejecutables pueden quedar en
una carpeta como `build_semana0/Debug`.

#### Limpiar artefactos

```bash
rm -rf build_semana0
rm -rf Semana0/Proyecto0/Testing
rm -rf Semana0/ejercicios0_out
```

#### Validación de documentación

```bash
grep -RIn "====\|—\|→" Semana0/*.md Semana0/lecturas/*.md Semana0/avanzado/*.md || true
```

El comando anterior ayuda a encontrar separadores o símbolos que no se quieren en
la documentación del curso.

### Fase 0.4: CMake moderno y CTest

#### Validación normal

```bash
rm -rf build_semana0
cmake -S Semana0/Proyecto0 -B build_semana0
cmake --build build_semana0
ctest --test-dir build_semana0 --output-on-failure
```

#### Validación sin demos ni benchmarks

```bash
rm -rf build_semana0_min
cmake -S Semana0/Proyecto0 -B build_semana0_min -DBUILD_APPS=OFF -DBUILD_BENCHMARKS=OFF
cmake --build build_semana0_min
ctest --test-dir build_semana0_min --output-on-failure
```

#### Limpieza

```bash
rm -rf build_semana0 build_semana0_min
rm -rf Semana0/Proyecto0/Testing
```

### Fase 0.5: Sanitizers

#### Validación con ASan y UBSan

```bash
rm -rf build_semana0_san

cmake -S Semana0/Proyecto0 -B build_semana0_san \
  -DCMAKE_BUILD_TYPE=Debug \
  -DCC232_ENABLE_SANITIZERS=ON

cmake --build build_semana0_san
ctest --test-dir build_semana0_san --output-on-failure
```

#### Demos intencionalmente incorrectos

Estos programas están diseñados para producir reportes de sanitizer cuando se ejecutan manualmente.

```bash
./build_semana0_san/demo_asan_out_of_bounds
./build_semana0_san/demo_ubsan_overflow
./build_semana0_san/demo_iterator_invalid
```

#### Limpieza

```bash
rm -rf build_semana0_san
rm -rf Semana0/Proyecto0/Testing
```
